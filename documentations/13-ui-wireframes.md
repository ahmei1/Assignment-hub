# UI Wireframes — Assignment Hub

## 1. Login Page

### Purpose

Allow users to log into the system.

### Layout

* Centered authentication card.
* Application logo at the top.
* Welcome message below the logo.
* Login form in the middle.
* Register link at the bottom.

### Components

* Logo
* Heading
* Email Input
* Password Input
* Login Button
* Register Link

### User Actions

* Enter email
* Enter password
* Submit login form
* Navigate to registration page

---

## 2. Register Page

### Purpose

### Layout

* Registration form in the middle.
* Login link at the bottom.

### Components

* Logo

### User Actions
