Use Case: Create Assignment

Actor: Lecturer

Steps:

1-Lecturer logs in.
2-Lecturer opens dashboard.
3-Lecturer clicks "Create Assignment".
4-Lecturer enters assignment details.
5-System saves assignment.

Use Case: Submit Assignment

Actor:Student

Steps:

1-Student logs in.
2-Student opens assignment.
3-Student uploads file.
4-Student submits.
5-System records submission.
